# 📰 React News App

A simple React side project built to fetch and read the latest news using the GNews API. Created to practice React class components, API fetching, and state management.

**Live Demo:** [View the aop here](https://newsapp-8a3l73pm3-vishwark-ais-projects.vercel.app/)

## Features
* **Live news fetching** by category
* **Search functionality** to find specific articles
* **Infinite scrolling** for a seamless reading experience

## How to Run Locally
1. Clone the repo and run `npm install`.
2. Grab a free API key from [GNews](https://gnews.io/) and paste it into the fetch URL inside `News.js`.
3. Run `npm start`.
